# Darbo eiga

1. Sukurta svetainės struktūra ir trys HTML failai.
2. Sukurtas bendras CSS stilius (naudotas Flexbox blokams išdėstyti).
3. Įdėti privalomi elementai: lentelė, sąrašas, forma ir paveikslėliai.
4. Sutvarkyti paveikslėlių `alt` tekstai dėl prieinamumo.
5. Kodas patikrintas W3C validatoriais – klaidų nėra.
